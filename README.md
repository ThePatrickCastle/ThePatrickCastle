## Selected work

**Unsupervised Discovery of Chess Tactical Patterns** — *Python · scikit-learn · NetworkX*
An end-to-end pipeline that represents chess positions as labeled directed graphs and clusters 300,000 positions from the Lichess open database using PCA and Gaussian mixture models. The number of clusters is chosen by the Bayesian Information Criterion and validated against held-out game-phase and material metadata.

**Parameter Passing via Continuations** — *Research poster · Faculty of Sciences, UNAM*
Co-authored study of continuation-passing style and its implications for parameter passing in functional languages. Presented at the Faculty's Programming Languages and Computing Symposium; supervised by Dr. Karla Ramírez Pulido.

**Adogta — Pet Adoption Platform** — *React · Kotlin/Spring Boot · FastAPI · PostgreSQL*
Full-stack, multi-service web platform built by a team of six, including an LLM microservice for breed-temperament classification and a recommendation algorithm for adopter–animal compatibility over a normalized PostgreSQL schema.

**pixellogicLSB — LSB Steganography Tool** — *C · libpng*
Command-line tool for hiding and extracting messages inside PNG images via least-significant-bit steganography, using a custom binary container format with a full test suite.
